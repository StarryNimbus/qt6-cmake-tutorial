# Getting Started with CMake

## How to build
- cd ~/Documents/qt6-cmake-tutorial/intro
- cmake .
- cmake --build .

## How to run
- ./helloworld

## Trouble shooting
Occassionally will need to do a clean build.

` cmake --build . --target clean `
